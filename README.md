
# SpaceX - Clone
 A project about completed exploration missions, new launches, knowledge and expiration


## Install

Install the style using

```bash
  npm sass --watch sass/style.scss:css/style.css
```

## Stack

**Front-end:** HTML, Sass, CSS, Bootstrap
